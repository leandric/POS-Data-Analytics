# Pós-Graduação em Data Analytics — FIAP

Este repositório reúne **projetos, estudos, notebooks e exercícios aplicados** desenvolvidos durante a Pós-Graduação em **Data Analytics** pela **FIAP**, com foco em estatística, machine learning, big data, visualização, séries temporais e deployment de modelos.

O objetivo é consolidar os conceitos teóricos e práticos vistos ao longo do curso, servindo tanto como **portfólio técnico** quanto como **material de referência**.
